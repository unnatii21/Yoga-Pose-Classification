# Setup Instructions

## Prerequisites
- Python 3.10
- Webcam

## Installation

1. Clone the repository:
   git clone https://github.com/unnatii21/Yoga-Pose-Classification.git

2. Install required libraries:
   pip install -r requirements.txt

3. Download the Kaggle Yoga Poses dataset and place it in a folder called dataset/

## Running the Project

1. Open Yoga_Client.ipynb in Jupyter Notebook or VS Code
2. Run cells 1-6 to train the model
3. Run the final cell to start live webcam detection

## Poses Supported
- Downdog
- Goddess
- Plank
- Tree
- Warrior2
